1. Gdzie jest wstawiany arkusz wewnętrzny css?

    2. Arkusz wewnętrzny css jest wstawiany w tagu < head > pod podstacią < style > ... < / style >

  2. Składnia teoretyczna (w ramce, konieczne wcięcia)

    3. < head >
    (...)
    < style type="text/css" >
    /* < ![CDATA[ */
    selektor1 { cecha: wartość; cecha2: wartość2... }
    selektor2 { cecha: wartość; cecha2: wartość2... }
    (...)
    /* ]]> */
    < / style>
    (...)

  3. Jakie mogą być selektory?

    4. Może to być dowolny znacznik np < p > , < h > , < td > itp.

  4. Jakie mogą być cechy i wartości akrusza wewnętrznego css ?

    5. Jako "cecha" (w deklaracji stylu - powyżej) należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (opisane w kolejnych rozdziałach).
    Natomiast jako wyraz "wartość" wpisujemy dokładną wartość atrybutu. Zwróć uwagę, że jednemu selektorowi możemy nadać kilka atrybutów (cech). Są one wtedy rozdzielone średnikami.
